TECH NEWS WEBSITE

A simple and clean Tech News webpage designed with HTML, CSS, and JavaScript. The project displays trending tech stories with a navy–black theme and interactive Read More buttons.

FEATURES:

1. Interactive Read More buttons powered by callback functions
2. Responsive blog cards displayed horizontally
3. Key JavaScript Concepts Used:
   a.Anonymous functions
   b.Arrow functions
   c.Callback functions
   d.Higher‑order functions
